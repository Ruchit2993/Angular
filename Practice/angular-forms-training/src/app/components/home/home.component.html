Welcome to Angular Template-Driven Forms Training! 👋

This training project is designed to help you master Angular Template-Driven Forms and validation techniques. Each form demonstrates different validation scenarios and best practices that you'll encounter in real-world applications.

📚 What You'll Learn:

  • Template-Driven Forms Basics - Understanding the FormsModule and NgModel directive
  • Built-in Validators - Using required, minlength, maxlength, email, pattern, etc.
  • Form Validation States - Working with touched, dirty, valid, and invalid states
  • Custom Error Messages - Displaying meaningful validation feedback to users
  • Form Submission - Handling form data and preventing invalid submissions
  • Styling Form States - Visual feedback using CSS classes for validation states
  • Best Practices - Industry-standard patterns for form development

🎯 Training Forms:

📝 Registration Form

Learn comprehensive validation with username, email, password, age, and more. Includes pattern matching and custom validators.

🔐 Login Form

Master basic validation with email and password fields. Perfect for understanding form states and error handling.

📧 Contact Form

Explore advanced patterns like phone validation, URL validation, and textarea handling with character limits.

🚀 Getting Started:

  • Navigate through each form using the tabs above
  • Try submitting forms without filling required fields
  • Observe how validation messages appear
  • Notice the visual feedback (colors and icons)
  • Check how the submit button gets disabled for invalid forms
  • Review the source code to understand the implementation

💡 Tips for Learning:

  • Open browser DevTools to inspect form elements and their classes
  • Experiment by modifying validation rules in the code
  • Pay attention to when validation messages appear (on touch/blur)
  • Study how NgModel creates two-way data binding
  • Understand the difference between template-driven and reactive forms
